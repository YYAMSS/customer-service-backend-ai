# 客服后端接口文档

## 1. POST `/api/chat`

发送一条用户消息，返回机器人本轮回复。

### 请求体

```json
{
  "sender_id": "user_001",
  "text": "帮我查一下订单状态",
  "object": {
    "type": "order",
    "id": "ORDER_10001",
    "title": "订单 ORDER_10001",
    "attributes": {
      "source": "order_list"
    }
  },
  "message_id": "msg_001"
}
```

### 字段说明

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `sender_id` | `string` | 是 | 用户唯一标识 |
| `text` | `string \| null` | 否 | 用户文本消息 |
| `object` | `object \| null` | 否 | 用户结构化对象消息 |
| `message_id` | `string \| null` | 否 | 客户端消息 ID；不传时服务端自动生成 |

### `object` 字段

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `type` | `string` | 是 | 对象类型，如 `order`、`product` |
| `id` | `string` | 是 | 对象 ID |
| `title` | `string \| null` | 否 | 对象标题 |
| `attributes` | `object` | 否 | 扩展属性，默认空对象 |

### 请求约束

- `sender_id` 必填
- `text` 和 `object` 至少传一个
- 若传 `object`，则 `object.type` 和 `object.id` 必填

### 响应体

```json
{
  "sender_id": "user_001",
  "message_id": "msg_001",
  "messages": [
    {
      "text": "请告诉我你的订单号。",
      "object": null
    }
  ]
}
```

### 响应字段

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `sender_id` | `string` | 是 | 用户唯一标识 |
| `message_id` | `string` | 是 | 本次消息 ID |
| `messages` | `array` | 是 | 机器人本轮回复列表 |

### `messages[]` 字段

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `text` | `string \| null` | 否 | 回复文本 |
| `object` | `object \| null` | 否 | 回复对象，结构同请求中的 `object` |

### 状态码

- `200 OK`

## 2. GET `/api/chat/history`

获取指定用户的聊天历史。

### Query 参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `sender_id` | `string` | 是 | 用户唯一标识 |

### 请求示例

```http
GET /api/chat/history?sender_id=user_001
```

### 响应体

```json
{
  "sender_id": "user_001",
  "messages": [
    {
      "role": "user",
      "text": "帮我查一下订单状态",
      "object": null
    },
    {
      "role": "bot",
      "text": "请告诉我你的订单号。",
      "object": null
    }
  ]
}
```

### 响应字段

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `sender_id` | `string` | 是 | 用户唯一标识 |
| `messages` | `array` | 是 | 历史消息列表 |

### `messages[]` 字段

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `role` | `string` | 是 | `user`、`bot` |
| `text` | `string \| null` | 否 | 消息文本 |
| `object` | `object \| null` | 否 | 对象消息，结构同请求中的 `object` |

### 状态码

- `200 OK`
